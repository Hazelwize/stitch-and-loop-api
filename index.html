Welcome to the Stitch and Loop API

Feel free to use this API, I have tried to make it as easy as possible to use. Here are the standard URL paths to use.

  • /api : This will give you all of the results from the api
  • /api/name/{name} : Will filter the results by the name given
  • /api/type/{type} : Will filter the results into groups of
    • stitches
    • patterns
    • techniques

I hope that you find this API useful!
